Plugins for JIRA, Bugzilla and Trac

If your project uses any of the issue trackers mentioned above, you might want to take a look at the FreedomSponsors plugins for JIRA, Bugzilla and Trac.

"Sponsor this issue" button

Make a “Sponsor this issue” link by pointing it to “http://www.freedomsponsors.org/core/sponsor?trackerURL=<url>”
Where <url> is the “original” issue URL.
Examples:

  • Sponsor HQL Unions on Hibernate
  • Sponsor Google Apps Authentication for Jenkins

Rest API

We are building a public JSON-based REST API that you can use to pull data from FreedomSponsors. There's not much right now, but if you need additional services, please don't hesitate to open a feedback issue
